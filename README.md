# 📺 BrightTV Viewership Analytics

## Project Overview

This project analyses BrightTV viewership data to understand how people watch TV, which channels are the most popular, and when viewers are most active. The goal is to help the business improve customer engagement and increase subscriptions by using data-driven insights.

## Objectives

* Analyse viewer behaviour.
* Identify the most popular TV channels.
* Find peak viewing days and hours.
* Understand viewer demographics.
* Create an interactive dashboard to present key insights.

## Tools Used

* SQL – Data cleaning and analysis
* Microsoft Excel – PivotTables and charts
* Power BI – Dashboard creation and data visualisation
* Looker Studio (Data Studio) – Interactive dashboard
* Miro – Project planning
* Figma – Dashboard planning and wireframes
* GitHub – Project documentation and version control

## Project Process

1. Understand the business problem.
2. Explore the dataset.
3. Clean and prepare the data.
4. Analyse the data using SQL and Excel.
5. Create charts and dashboards.
6. Write insights and recommendations.
7. Document the project on GitHub.

## Dashboard Features

* Total Viewers
* Actual viewers
* Number of channels
* Avarage hour
* Total Watch Time
* Top 5 Most Viewed Channels
* Viewership by Month
* Viewership by Province
* Viewership by Gender
* Viewership by Age Group
* Viewership by Day of Week
* Viewership by Time of Day
* Peak Viewing Hours

## Key Insights

* March had the highest number of viewers.
* Most viewers watched TV during the evening.
* Some channels attracted significantly more viewers than others.
* Viewer activity changed depending on the day of the week.
* Certain provinces had higher viewership than others.
* Different age groups showed different viewing patterns.

## Recommendations

* Schedule popular programmes during peak viewing hours.
* Promote less popular channels to increase engagement.
* Create targeted marketing campaigns for different age groups and provinces.
* Focus customer retention efforts during periods of lower activity.
* Continue monitoring viewing trends to support business decisions.
